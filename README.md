# 💰 Kişisel Gider Takip Sistemi

Bu proje, harcamalarınızı kategorize ederek yerel bir veritabanında (txt dosyası) saklamanızı sağlayan bir Python uygulamasıdır.

## 🛠 Teknik Özellikler
- **Dosya İşlemleri:** Verileri kalıcı olarak saklamak için File I/O kullanıldı.
- **Hata Yönetimi:** Geçersiz veri girişlerine karşı 'try-except' blokları ile koruma sağlandı.
- **Algoritma:** 'While' döngüsü ve 'if-elif-else' yapıları ile interaktif bir menü oluşturuldu.

## 🚀 Nasıl Çalıştırılır?
1. Python yüklü olduğundan emin olun.
2. `gider_takip.py` dosyasını çalıştırın.
3. Menüden yapmak istediğiniz işlemi (Ekle, Listele, Hesapla) seçin.
